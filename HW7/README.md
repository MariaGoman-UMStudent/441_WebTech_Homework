This is the final result of my memory game. Overall, I am pleased with how it turned out, but I should have done a few things differently. For instance, I should have put the placeholders for the names and age inside of a div container because I wanted them to be centered, which would improve the overall aesthetic and design. What's more important to me right now is learning how to make my websites functional. Thankfully, the website works effectively. I try not succumb to the temptation of using A.I, but it has helped me a lot with the debugging proccess. I ran into a few problems where the buttons wouldn't work, and the use of A.I helped me find the issues easily. In a way, it's a huge relief for me because I don't have to spend long periods of time googling for answers with little to no success. What I learned from the debugging process was that the problems usually stemmed from me forgetting to call a function. This whole experience really highlighted the utter importance of functions being called or else they will not work properly.
