I built a page that loads and displays data from a local JSON file using jQuery and AJAX. I used a dataset from JSONlint about European countries with info like population, capital, and EU membership.I used $.getJSON() to load the data and $.each() to loop through it and then I created cards to display each country in a clean and readable way. For the plugin, I made a highlightOnHover function using $.fn that adds a hover effect to each card. One issue I ran into was that AJAX wouldn’t load my JSON when I opened the file normally (file://). I learned that it only works through a server, so I used Live Server to fix it. I used class examples, resources from Google, and some AI to help me understand how everything connects.
