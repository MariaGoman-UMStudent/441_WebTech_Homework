It took me some time to decide on a theme for this project. I ultimately chose a dream-like concept because I’ve always been drawn to its soft aesthetic and the sense of calm it creates. I wanted my poster to feel like drifting through a dream, with gentle motions and transitions (Or at least, I attempted to). I found working with jQuery challenging, as it’s something I’ve struggled with in the past. However, this project helped me to understand it a little bit better, though I feel like this project still needs some polishing. Honestly, I had different ideas of how this project would turn out, but I did the best I could. I did use a little bit of Generative AI for debugging, and to help structure my code and organize my animation systems.
